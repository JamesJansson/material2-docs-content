API reference for Angular Material legacy-progress-spinner

import {MatLegacyProgressSpinnerModule} from '@angular/material/legacy-progress-spinner';

Directives

MatLegacyProgressSpinner

<mat-progress-spinner> component.

Selector: mat-progress-spinner mat-spinner

Exported as: matProgressSpinner
Properties
Name Description
@Input()

color: ThemePalette

Theme color palette for the component.
@Input()

diameter: number

The diameter of the progress spinner (will set width and height of svg).
@Input()

mode: LegacyProgressSpinnerMode

Mode of the progress circle
@Input()

strokeWidth: number

Stroke width of the progress spinner.
@Input()

value: number

Value of the progress circle.

defaultColor: ThemePalette | undefined

Default color to fall back to if no value is set.

Interfaces

MatLegacyProgressSpinnerDefaultOptions

Default mat-progress-spinner options that can be overridden.

Properties
Name Description

color: ThemePalette

Default color of the spinner.

diameter: number

Diameter of the spinner.

strokeWidth: number

Width of the spinner's stroke.

Type aliases

LegacyProgressSpinnerMode

Possible mode for a progress spinner.

type LegacyProgressSpinnerMode = 'determinate' | 'indeterminate';

Constants

MAT_LEGACY_PROGRESS_SPINNER_DEFAULT_OPTIONS

Injection token to be used to override the default options for mat-progress-spinner.

const MAT_LEGACY_PROGRESS_SPINNER_DEFAULT_OPTIONS: InjectionToken<MatLegacyProgressSpinnerDefaultOptions>;
